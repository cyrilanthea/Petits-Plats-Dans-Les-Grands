Petits Plats Dans les grands

Le site est composé de 5 pages dont l'accueil, les commandes,
le points de retraits, les cours de cuisine et à propos.

- La page d'Accueil illustre le titre et le slogan du restaurant.
- La page des commandes est constitué de 2 Entrées,2 plats,
  2 desserts et un menu végétarien.
- La troisième page contient les points de retraits et
  une image de carte qui regroupent tout les points. Nous voulions
  insérer une vrai carte mais ils nous manquent les techniques nécessaires
  pour réaliser cette tâche.
- L'avant dernière page est dédier pour le cours de cuisine, on propose
  3 cours thématiques avec des images et en affichant les horaires concerné.
- Dans la page de à propos, on mets en avant les valeurs, les producteurs de référence du restaurant .
